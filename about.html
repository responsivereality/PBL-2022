Project Print is an initiative to excite people about 3D printing and its capabilities, especially relating to science. We hope to get more people interested and perhaps take part in the use of this amazing machine. In total, this project took almost 69 hours and 870 grams of plastic, costing us only $24 worth of filament. We hope to show how cost-efficient, yet incredible 3D printing can be, especially considering its the things that can be achieved with it.

As an entry-level printer, we would recommend the Ender 3 and its variants, the main of which can be found here. All of them are priced roughly at $350 including shipping. I (Dylan) personally use the Prusa i3 MK3s+ which costs around $1500 with the kit or $1800 preassembled. This is considering international shipping and importing fees.

To tie our project together, I (Ezra) decided to fully develop, code and host a website on the Internet, in fact this website that you are reading from right now. Although it was an arduous task, I felt that some sort of showcase was needed for this cool project, so this was our decision. I utilised the coding languages HTML, CSS and JavaScript to create an incredibly interactive website for everybody to see. Please enjoy Ezra and Dylan's PBL 2022 Project.
